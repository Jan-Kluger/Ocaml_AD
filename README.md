# Algorithms & Data structures in Ocaml
Practicing Algorithms and datastructures in purely functional OCaml. 

This project contains
- Stacks
- Queues
- Quicksort
- Mergsort
- Binary Heap
- Avl Trees
- ab Trees
- Hashing with chaining
- Hashing with linear probing
- Graphs
- BFS
- DFS

Working on:
- Dijkstras
- Bellman-Ford
- Floyd-Warshall
- Johnson-Algorithm
- Kruskal
- Prim
